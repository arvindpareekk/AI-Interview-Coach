# 🤖 AI Interview Coach

An **AI-powered interview preparation platform** that helps candidates practice technical interviews by analyzing resumes and generating intelligent interview questions.

Built to simulate a **real interview experience** using AI and provide meaningful insights to improve candidate performance.

---

## 🚀 Features

* 📄 **Resume Analysis** – Upload your resume (PDF) and extract key information.
* 🤖 **AI Question Generation** – Generates interview questions based on resume content.
* 🧠 **AI Evaluation** – Evaluates candidate answers and provides feedback.
* 📊 **Score & Insights** – Gives performance scores and improvement suggestions.
* 💻 **Interactive UI** – Built with Streamlit for a smooth user experience.

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/arvindpareekk/AI-Interview-Coach.git
```

Move into project directory

```
cd AI-Interview-Assistant
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run app.py
```

## 🎯 How It Works

1️⃣ Upload your resume (PDF)
2️⃣ AI analyzes resume content
3️⃣ AI generates personalized interview questions
4️⃣ Candidate answers the questions
5️⃣ AI evaluates responses and provides feedback
